常见存储器

Author: xyy Date: 九月 28, 2019  16:08:08 Category: STM32F429

存储器种类


易失性与非易失性：掉电之后数据是非还存在

RAM

RAM是“Random Access Memory”的缩写，被译为随机存储器

特性 动态随机存储器DRAM(Dynamic RAM) 静态随机存储器SRAM(Static RAM)
结构
存取速度 较慢 较快
集成度 较高 较低
生产成本 较低 较高
是否需要刷新 是 否

是否需要刷新是根据结构决定的： DRAM通过电容的电压来判断是1/0，但电容会自己放电导致出错，所以需要刷新不断充电是其稳定。（有一个值 高于这个值是1 否则是0）

同步/异步存储器

判断是否有时钟信号线，有同步，无异步。

常见的DRAM都采用同步方式，称为SDRAM(Synchronous DRAM)。

种类 特点
普通SDRAM 在上升沿时同步数据
DDRII SDRAM 在上升沿及下降沿都同步数据，时钟极限频率800MHz
DDRIII SDRAM 在上升沿及下降沿都同步数据，时钟极限频率1600MHz

常见的SRAM都采用异步方式，它们被直接称为SRAM。

ROM

ROM是“Read Only Memory”的缩写，意为只能读的存储器。
(随着技术的发展已经可以写了，只是名字没改，现在一般用于指代非易失性半导体存储器)

种类 特点
MASK ROM 出厂时固化，不可修改
OTPROM 用户可写入一次，之后不可修改
EPROM 可重复擦写，需要使用专用紫外线照射设备擦除
EEPROM 可重复擦写，电擦除，使用方便

FLASH

FLASH存储器又称为闪存，它也是可重复擦写的储器，部分书籍会把FLASH存储器称为FLASH ROM，但它的容量一般比EEPROM大得多，且在擦除时，一般以多个字节为单位

坏块: 当编程/擦除这个块时，不能将某些位拉高.
XIP:CPU能否直接运行上面存储的程序

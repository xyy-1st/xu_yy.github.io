可变参数

Author: xyy Date: 十一月 7, 2019  21:11:38 Category: C/C++ Pageviews: loading

写法 形参表最好写 …

如int func(int num, … ) {},int func(char a,int num, … ) {}
一般省略号之前的那个参数是 int，代表了要传递的可变参数的总数。

  1. 定义一个函数，最后一个参数为省略号，省略号前面可以设置自定义参数。
    int func(int num, … ){}
  2. 在函数定义中创建一个 va_list 类型变量，该类型是在 stdarg.h 头文件中定义的。
    va_list vl;
  3. 使用…前一个形参 和 va_start 宏来初始化 va_list 变量为一个参数列表。
    va_start(vl, num); 此时vl指向 …的第一个
  4. 使用 va_arg 宏和 va_list 变量来访问参数列表中的每个项。
    int result = va_arg(vl, int)；
    va_arg()返回vl当前指的值，并将vl指向下一个。 第二个参数写…的类型
  5. 使用宏 va_end 来清理赋予 va_list 变量的内存。
    va_end(vl);

头文件stdarg.h 中相关的宏定义

  1. typedef char * va_list
  2. #define va_start ( ap, v ) ( ap = (va_list)&v + _INTSIZEOF(v) )
  3. #define _INTSIZEOF(n) ((sizeof ( n ) + sizeof ( int ) – 1 ) & ~( sizeof( int ) – 1 ) )
  4. #define va_arg(list, mode) ((mode *)(list =(char *) ((((int)list + (builtin_alignof(mode)<=4?3:7)) &(builtin_alignof(mode)<=4?-4:-8))+sizeof(mode))))[-1]
  5. #define va_end ( list )

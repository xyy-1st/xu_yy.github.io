向量相加

Author: xyy Date: 十一月 21, 2019  12:10:18

cblas_?axpby ？是什么

？表示数据是什么类型

  • s 单精度实数
  • d 双精度实数
  • c 单精度复数
  • 双精度复数

    定义

    函数定义为：
    void cblas_saxpby (const MKL_INT n, const float a, const float x, const MKL_INT incx,const float b, float *y, const MKL_INT incy);
    得到的结果是:
    y = a    x + b*y

    参数说明

  1. const MKL_INT n 向量的长度
  2. const float a 代表上式中的a
  3. const float *x 向量x
  4. const MKL_INT incx 表示x的增量(我也不懂 师兄告诉我写1)
  5. const float b 代表上式中的b
  6. float *y 向量y（同时结果存在这里）
  7. const MKL_INT incy 表示y的增量(我也不懂 师兄告诉我写1)

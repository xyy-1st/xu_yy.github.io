那么我们也可以定义一个函数型的指针 让它指向函数 这就是函数指针了
写法： 函数返回值类型 ( * 指针变量名) (函数参数列表);
int ( *fun)(int x, int y);
int：是返回的类型
( *fun)：指针名字
(int x, int y)：参数
fun 的类型是 int ( * )(int ,int)

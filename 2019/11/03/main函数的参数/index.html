main函数的参数

Author: xyy Date: 十一月 3, 2019  18:51:04 Category: C/C++ Pageviews: loading

int main (int argc, char *argv[], char *envp[])

int argc 是表示你在命令行 下输入命令的时候，一共有多少个参数。
char* argv[] 是记录你所输入的参数。
char *envp[] 存的是环境变量（很少用到）

D:>VSproject\Project1\x64\Debug\Project1 you xiu de ren

当我们切换目录之后在执行是的结果：

数学函数

Author: xyy Date: 一月 4, 2020  20:18:09 Category: C语言 Pageviews: loading

头文件说明

<math.h> 是C语言中数学函数库
是C++中对应相同作用的头文件
当然C++中两个头文件都可以使用

求绝对值函数

  1. int abs(int x)
    abs(-4) = 4
    abs(-3.14) = 3

  2. long labs(long x)
    labs(-41576) = 41576

  3. double fabs(double x)
    fabs(3.14) = 3.14

int 与 long 的区别
早期int是2字节，现在已经发展到4字节
而long一直是4字节
所以现在int与long没区别
long long 是8字节

三角函数

  1. 正弦函数 dobule sin(double x)
    sin(3.1415926535/2) = 1
  2. 余弦函数 dobule cos(double x)
    cos(3.1415926535/2) = 0
  3. 正切函数 dobule tan(double x)
    tan(3.1415926535/4) = 1
  4. 反正弦函数 dobule asin(double x)
    asin(0) = 0
  5. 反余弦函数 dobule acos(double x)
    acos(0) = 1.570796
  6. 反正切函数 dobule atan(double x)
    atan(0) = 0

其中sin,cos,tan与数学中的sin,cos,tan一一对应
asin,acos,atan与数学中的arcsin,arccos,arctan一一对应

计算ex

double exp(double x)
exp(1) = 2.71828

计算ln x

double log(double x)
log(2.17828) = 0.999999

计算 log10 x

double log10(double x)
log10(10) = 1

计算xy

double pow(double x,double y)
pow(2.0,3.0) = 8

计算根号x

double sqrt(double x)
sqrt(4) = 2

取舍函数

向下取整double floor(doblue x)

floor(2.8) = 2
floor(-2.8) = -3
floot(2) = 2
floot(-2) = -2

向上取整double ceil(double x)

ceil(2.8) = 3
ceil(-2.8) = -2
ceil(2) = 2
ceil(-2) = -2

注意：返回值都是double类型,而不是int类型
原因：double的范围是-1.710^308 ~ 1.710^308
　　　int的范围是-2147483648 ~ 2147483647
　　　double的范围比int的范围大.

实现四舍五入

float a;
int b = (int)(a + 0.5);

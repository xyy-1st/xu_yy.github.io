shell

Author: xyy Date: 七月 1, 2020  9:03:40 Category: Linux

shell脚本一般后缀名为 .sh
开头写#!/bin/bash　　　注释使用/bin/bash解释器

启动方式

  1. 当程序运行 ./hello.sh
  2. 指定解释器 /bin/bash hello.sh
  3. source和.　　source hello.sh 和 . hello.sh

变量

定义变量（不能有空格）

  1. var=value　　　　适用于没有空格和Tab时
  2. var=’value’　　　‘’里面是什么就是什么
  3. var=”value”　　　“” 里面如果使用了其他变量，会替换为变量的值

使用变量

  1. $var
  2. ${var}　　　　为了与后面的其他内容区分(因为不能使用空格)

将命令的结果赋值给变量

  1. var=`command`　　　　注意是反引号,在Tab键上面
  2. var=$(command)

删除变量

unset var

特殊变量

变量 含义
$0 shell脚本文件名
$n 第n个参数
$# 参数的个数
$* 所以参数
$@ 所以参数
$? 获取上一个命令的退出状态活函数返回值
$$ 进程ID

$* 与 $@的区别见for in循环例5

字符串拼接

直接写再一起
例：var=$(pwd)aaa

读取从键盘输入的数据 read

例：

  1. read a
  2. read -p “input a:” a 加入提示信息 input a:

对整数进行数学运算 (())

例:
a=2
b=3
sum=$(($a+$b)) 或 sum=$((a+b))

逻辑与/或(同C语言)

command1 && command2
当command1 为真时才执行command2
command1 || command2
当command1为假时才执行command2

检查某个条件是否成立

test expression 和 [expression]

管道

command1 | command2
command1的结果作为command2的输入
注意：command1必须正确

if语句

case in语句

for in 循环

while 循环

函数
